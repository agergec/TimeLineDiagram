SIP Log Parser — Help

Getting Started

Input
Paste Genesys SIP Span Quick Flow output and click Parse. Supports SIP messages, Events, and Requests.
Auto-load
If only one saved log exists, it loads automatically on page open.
Save / Load
Save parsed logs to browser storage (up to 20). Access via Saved button.

Views

SIP Span
All messages in chronological order with full detail columns.
Kazimir
Events & Requests arranged in a grid by Device Number (DN). Each DN gets its own column.

Filters

Type filters
SIP, Events, Requests — toggle each message type on/off.
CID tags
Click a CID tag to enable/disable it. Use All / None for bulk toggle. Tags are disabled (greyed) when SIP filter is off.

Delta Modes

Per-CID
Time since the previous message within the same CID.
Cross-CID
Time since the previous visible message across all filtered CIDs. Recalculates when CIDs are toggled.
Both
Shows both delta columns side by side.

Columns (SIP Span)

Time
Absolute timestamp HH:MM:SS.mmm
Elapsed
Milliseconds since the first message in the log.
Δ Delta
Per-CID or Cross-CID delta with speed indicator dots.
Type
Badge: SIP, EVT, or REQ
Dir
Arrow showing direction — incoming or outgoing.
Method
SIP method (INVITE, ACK, BYE, etc.) or Event/Request type name.

Speed Dots

Fast
< 100 ms
Medium
100 – 499 ms
Slow
≥ 500 ms

Bookmarks

Click a row
Toggle bookmark on that message. Bookmarked rows are highlighted.
Bookmark bar
Appears in the header showing bookmarked times with step deltas (→ +Xms) and cumulative time (ΣXms).
Persistence
Bookmarks auto-save with the log and restore on load. They persist across SIP Span / Kazimir tab switches.
Clear
× button removes all bookmarks.

Info Bar

Stats
Shows ConnID, total count, SIP/Events/Requests breakdown, error count, CID count, INVITE count, and total time span.

Export

Import to Timeline
Extracts INVITE→ACK pairs from enabled CIDs and creates a timeline diagram in the Timeline Editor.
← Timeline Editor
Opens the Timeline Diagram Editor app.

Other

Theme
Toggle dark / light mode. Preference is saved.
Esc
Closes any open modal.
